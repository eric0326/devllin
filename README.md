# 德文琳金属工艺品网站 (my.gddevllin.com)

静态站,部署在 Cloudflare Pages,GitHub 自动同步。

## 文件
- `index.html` — 首页(自包含,内联 CSS + SVG)

## 部署
1. 推送到 `main` 分支
2. Cloudflare Pages 自动构建部署

## 本地预览
双击 `index.html` 即可。

## 后续添加
- `about.html` `blog.html` `contact.html` — 子页面
- `products/` — 各产品详情页
- `assets/` — 真实产品图(替换 SVG 占位)
